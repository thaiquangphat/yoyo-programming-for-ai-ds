Key Findings

🏠 Tabular Insights

  • Feature Engineering Impact: 14 engineered features created (HouseAge, TotalSF, TotalBath, etc.) improving model performance 3-5%
  • Log Transformation Critical: Target (SalePrice) log transformation essential for normal distribution and linear relationships
  • Gradient Boosting Dominance: Ensemble models (LightGBM, XGBoost, GradientBoosting) outperform linear models by 13-14% RMSLE
  • Top Features: OverallQual, GrLivArea, TotalSF, GarageArea, TotalBath consistently ranked highest
  • Multicollinearity Handled: L2 regularization (Ridge) and feature selection (Lasso) manage GarageCars/GarageArea correlation

🎯 Multimodal Insights

  • Fusion Superiority: Learned fusion MLP achieves 17.7% improvement over best single modality (Micro-F1: 0.5412 vs 0.4598)
  • Modality Balance: Text (DistilBERT: 0.4598) and Image (ResNet-50: 0.4621) similarly informative, complementary signals in fusion
  • Transfer Learning Power: Fine-tuned DistilBERT/ResNet-50 outperform TF-IDF/classical ML by 8-10% F1 score
  • Learned Fusion Best: MLP fusion (0.5412 F1) outperforms simple averaging (0.5187) and concatenation (0.5034)
  • Class Imbalance Managed: BCEWithLogitsLoss with pos_weight, balanced class weights, early stopping prevent overfitting

Methodology

📋 Tabular: House Prices

  1. Data Cleaning: Handle missing values via domain logic (absence = 'None' for categorical, median/mode for numeric)
  2. Feature Engineering: 14 engineered features (age, ratios, binary flags) + 134 encoded features
  3. Transformation: Log1p applied to 22 skewed features; target-encoding for high-cardinality variables
  4. Model Training: 8 models evaluated with 5-fold CV: Ridge, Lasso, ElasticNet, DecisionTree, RandomForest, GradientBoosting, XGBoost, LightGBM
  5. Evaluation: RMSLE metric; LightGBM selected as best (RMSLE: 0.1055)

🎯 Multimodal: COCO Classification

  1. Data Preparation: Top-60 categories selected; 70/15/15 train/val/test split; 3.95 labels/image avg
  2. Text Models: TF-IDF + LR/SVC (classical), Fine-tuned DistilBERT (transfer learning)
  3. Image Models: ResNet-50 features + LR (frozen), Fine-tuned ResNet-50 (end-to-end)
  4. Fusion Strategies: Early fusion (concatenation), Late fusion (probability averaging), Learned fusion (MLP)
  5. Evaluation: Multi-label F1 (micro/macro/sample), Hamming loss, Precision, Recall
