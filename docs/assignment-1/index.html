About Assignment 1

This assignment explores a multi-modal dataset combining images and textual descriptions. The Multi-modal EDA report provides comprehensive analysis including:

  • Dataset Overview: Summary statistics and data quality metrics
  • Image Analysis: Resolution, aspect ratios, color profiles, and brightness
  • Text Analysis: Vocabulary, POS tags, n-grams, and embeddings
  • Cross-modal Analysis: Object-word alignment and category co-occurrence
  • Advanced Insights: Bias analysis, difficulty metrics, and topic modeling
  • Interactive Visualizations: t-SNE embeddings, category galleries, and more
